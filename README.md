# Marketing Campaign Dashboard 📊

An end-to-end Power BI dashboard project analyzing 200,000 marketing campaigns across multiple companies, channels, and audience segments using a 2021 dataset.

## 📁 Project Structure
- **6 Dashboard Pages** — Executive Overview, Company Performance, Channel & Campaign Analysis, Performance Trends, Audience Insights, Advanced Analytics
- **Data Cleaning** — Handled in Power Query (currency formatting, data type conversion, custom columns)
- **DAX Measures** — KPI cards, dynamic metrics, top-N calculations

## 🛠️ Tools Used
- Power BI Desktop
- Power Query (M Language)
- DAX

## 📌 Key Insights
- Email channel drives highest spend ($420.9M) and clicks (18.49M)
- October recorded peak campaign activity and clicks
- French-speaking audiences show highest conversion rate (8.03%)
- Foodies segment leads in total clicks (22.08M)
- All 5 companies perform within a narrow ROI band (4.99–5.01)
